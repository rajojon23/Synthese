
-Dézippez le fichier zip contenant le jeu




Avec Apache:
-Mettre le dossier dézippé vers le dossier htdocs de votre Apache 
-Ouvrez le fichier index.php dans votre serveur localhost 

Avec Wamp:
-Mettre le dossier dézippé vers le dossier www de votre wamp
-Ouvrez le fichier index.php dans votre serveur localhost

Installer wamp ici:
http://www.wampserver.com/ 

L'utilisation de Google Chrome comme navigateur web est conseillé fortement.

Pour les développeurs:
-Tous les fichiers nécéssaires sont dans html 
-Les fichiers javascript utilisés se trouvent dans le dossier js
-Les images se trouvent dans images
-La bande sonore et les effets se trouvent dans le dossier sound
-Les fichiers css se trouvent dans le dossier css
-Les polices se trouvent dans le dossier font
